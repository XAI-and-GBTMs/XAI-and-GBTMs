# SHAP and Model Diagnostics Plots

This directory contains the visual outputs generated by the XGBoost and SHAP analysis script. These plots serve as the core evidence for our NeurIPS-style report (specifically referenced in Section 3.5 and Appendix E).

## Contents

### 1. Explainability (SHAP)
* `shap_bar.png` & `shap_waterfall.png`: Global and local feature importance for the full 17-feature baseline model.
* `reduced_shap_bar.png` & `reduced_shap_waterfall.png`: Global and local feature importance for the pruned 12-feature model.

### 2. Feature Pruning
* `feature_elimination_rmse.png`: The sequential feature elimination curve. This proves that dropping the 5 lowest-ranked SHAP features minimizes our RMSE to 207.6.
* Similarly, `reduced_feature_elimination_rmse.png` is the sequential feature elimination curve for the pruned model.

### 3. Model Diagnostics
* `actual_vs_predicted.png`: Shows the tight clustering of our predictions along the diagonal for the test set.
* `residuals_plot.png`: Demonstrates that our errors are relatively well-distributed around zero without severe heteroscedasticity.

### 4. Exploratory Data Analysis
* `correlation_matrix.png`: The Pearson correlation matrix proving the extreme collinearity ($r=0.91$) between Ambient Temperature and Dew Point Temperature.
